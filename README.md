# 🎰 Discord カジノBot

PVE(スロット・チンチロ)と PVP(丁半・テキサスホールデム・5カードドロー)で遊べる
Discord カジノBot。コマンドは全て日本語、操作はパネル(ボタン)中心。チップ経済は
Bot内で完結し、インフレ対策・射幸性演出・管理ダッシュボードを備える。

## ゲーム
| ゲーム | 種別 | 概要 |
|---|---|---|
| 🎰 スロット | PVE | 3リール。段階表示・ニアミス演出・プログレッシブJP搭載 |
| 🎲 チンチロ | PVE | 親(Bot)と役勝負。ピンゾロ/シゴロ/ヒフミ等 |
| ⚂ 丁半 | PVP | 丁/半に賭けて当てた側が山分け |
| 🃏 ホールデム | PVP | ベットラウンド付きテキサスホールデム(サイドポット対応) |
| 🎴 ドローポーカー | PVP | 5カードドロー、1回交換 |

## 主なコマンド
- `/カジノ` … メインパネルを表示(ここから全ゲーム起動)
- `/残高` `/デイリー` `/送金` `/ランキング`
- `/スロット` `/チンチロ` `/丁半` `/ホールデム` `/ドロー`(パネルを使わず直接起動も可)
- `/ルール [ゲーム]` … 初心者向けの遊び方・役表・用語
- `/管理パネル` … 管理ダッシュボード(統計・設定変更)/ 管理者のみ
- `/管理 付与|没収|セット|凍結|解凍|監査|設定|統計|リロード` … 管理者操作

## 設計の要点
- **パネル化**: 入口は `/カジノ` のハブパネル。各ゲームはボタン→モーダルで賭け額入力。
  ハブパネルは永続View(再起動後もボタンが生きる)。
- **チップ経済**: SQLite。全チップ移動を `tx_logs` に記録(監査用)。新規ユーザーには
  初期残高を自動付与。
- **公平性**: 乱数は `secrets.SystemRandom`(暗号論的乱数)。
- **ハウスエッジ可変**: スロット配当はペイテーブルの素RTPを `1 - slot_house_edge` に
  毎回スケールするので、管理者がエッジを変えれば実効RTPも追従。
- **射幸性(脳汁)**: リール段階表示・リーチ/ニアミス表示・連勝ストリーク・
  プログレッシブJP。**配当倍率は上げず演出で射幸性を作る**方針(インフレと両立)。
- **インフレ対策(マイルド)**: デイリー残高減衰 / PVEハウスエッジ(負け分は消滅) /
  PVPレーキ(勝ち分から徴収して消滅) / 保有税(初期OFF)。JPはベットからの再分配で
  インフレ中立。`/管理パネル` で総供給量・シンク・資産上位を可視化。
- **全チューニング値は `settings` テーブル**にあり、`/管理 設定` や管理パネルから
  実行中に変更可能(再起動不要)。

## ディレクトリ
```
bot.py            起動・Cogロード・コマンド同期
config.py         環境変数(起動時不変の設定)
core/             純粋ロジック: deck/dice/hand/economy/match
db/               schema.sql と DAO(残高・設定・JP・統計)
cogs/             各ゲーム・経済・ハブ・ルール・管理の Cog
ui/               共通UI(Embed/ベットモーダル/権限)
deploy/           systemd ユニットとVPS手順
tests/            Discord非依存のロジック検証
```

## ローカル起動(Windows)
```powershell
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
copy .env.example .env   # DISCORD_TOKEN と ADMIN_IDS を記入
.venv\Scripts\python bot.py
```
開発中は `.env` の `DEV_GUILD_ID` に自分のサーバーIDを入れると、コマンドが即時反映される。

## テスト
```powershell
.venv\Scripts\python tests\test_core.py
.venv\Scripts\python tests\test_holdem.py
```

## 主な設定キー(`/管理 設定` で変更)
| キー | 既定 | 説明 |
|---|---|---|
| `starting_balance` | 1000 | 初期残高 |
| `min_bet` / `max_bet` | 10 / 100000 | ベット範囲 |
| `daily_base` | 1000 | デイリー基本額 |
| `daily_decay_enabled` | 1 | デイリー残高減衰 |
| `slot_house_edge` | 0.05 | スロットのハウスエッジ |
| `chinchiro_house_edge` | 0.05 | チンチロのハウスエッジ |
| `pvp_rake` | 0.03 | PVP手数料 |
| `jackpot_contrib` | 0.01 | ベットからJPへ積む割合 |
| `holding_tax_enabled` | 0 | 保有税(退蔵対策、初期OFF) |
